# 1. Objetivo

Esse diretório tem como objetivo documentar o processo de construção da modelagem de dados dos voos operados no Brasil, desde o modelo conceitual ao físico.

# 2. Disposições Gerais

**Fonte de Dados:** API ANAC

**Arquitetura:** Utilizaremos a Arquitetura Medalhão e em Cada Camada teremos sua devida modelagem.

# 3. Infos Gerais Sobre Cada Camada

**Bronze:** Dado raw em wide table

**Silver:** Modelagem dimensional com star schema

**Gold:** Data Mart
